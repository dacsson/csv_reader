# Как запустить 

`docker compose up`

или 

`docker compose up --build --force-recreate`

# Где находится код

csv_viewer - бэкенд NextJS + Swagger (TypeScript) | порт 5000

csv_viewer_front - фронтенд ReactJS + MUI (TypeScript) | порт 8085

# Swagger

Открыть можно по пути localhost:5000/api
